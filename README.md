## Mr. Sun

this app shows what time the sun wakes up and what time it goes to sleep in one city. the main screen has a big card with that day's sunrise and sunset time, and how long the sun is out. there is also a sideways row with the next 5 days, so you can see the sun times for the week.

**potential features**

check sunrise and sunset for other cities

save favorite locations

rate sunset with score

show best time to see

save watched sunsets
